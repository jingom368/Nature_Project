<h1>Google Analytics 분석</h1>
<p>· 프로젝트 참여 - 무용예술 기반 초기스타트업 '움트'의 두 번째 프로젝트</p>
<p>· 팀 구성 - 기획자 1명, 디자이너 1명, 개발자 1명</p>
<p>· 상세 역할 - 개발(BE & FE & DB) 설계 및 구현, 비즈니스 운영 지원</p>

<p>· 핵심 성과 : 총 300명의 공연 관람객 유치와 14명의 후속 프로젝트 파트너로부터 컨택을 받음</p>
<p>· 마케팅 전략 : Google Ads와 Instagram을 이용한 광고 진행, Google Analytics로 웹사이트 유입 분석하여 마케팅 효과 측정</p>

· 개발 성과 :
- 백엔드와 프론트엔드 설계 및 구현 완료
- Django를 활용하여 필요한 기능들을 구현
- 반응형 웹사이트 구현으로 모바일 환경에서의 접근성 향상
- 관람 신청에 관한 CRUD 기능을 구현하여 사용자 편의성 제공
- 공연에서 관람객과 함께하는 프로그램에서 웹페이지를 활용
  
<div>한 달 동안 약 1,000명의 방문자를 기록하였으며, 공연 당일에는 300명의 관람객을 유치하였습니다.</div>
<div>방문자와 관람객이 전 연령층에서 고르게 분포하였음을 확인하였습니다. 이를 통해 프로젝트의 전반적인 가능성을 확인하였습니다.</div>
